### 4.3 파생변수 현황

총 **21개** 파생변수가 백엔드 6개 DTO + 프론트엔드 3곳에 구현되어 있습니다.

| 카테고리 | 파생변수 | 수 | 산출 위치 |
|---------|---------|---|---------|
| 일별 효율 분석 | daysSinceRelease, audiPerScreen, audiPerShow, screenShare | 4개 | 서버 (DerivedStatsDto) |
| 영화 추적 | daysSinceRelease, weekNumber, audiPerScreen, audiPerShow | 4개 | 서버 (MovieTrackingDto) |
| 트렌드 집계 | period(월/계절), totalSales, totalAudience, movieCount, topMovie, avgScreens | 7개 | 서버 (WeeklyTrendDto) |
| 역대 순위 | maxAudiAcc, maxSalesAcc, rank | 3개 | 서버 (AllTimeRankingDto) |
| TMDB 통계 | avgRating, rating(장르별), 평균평점(연도별) | 3개 | 프론트엔드 |

상세 산출식 및 검증 결과는 **파생변수정의서** 참조.

## 5. 기능 분석

### 5.1 TMDB 영화 대시보드 (5개 탭)

| 탭 | 시각화 | 차트 유형 | 데이터 소스 |
|----|--------|----------|-----------|
| 랭킹 | Top N 영화 순위 리스트 | 테이블 + 카드 | movies (필터 적용) |
| 장르 | 장르별 영화 수 / 평균 평점 | 수평 BarChart | movies → 장르 그룹핑 |
| 트렌드 | 연도별 개봉 수 + 평균 평점 | 이중축 ComposedChart | movies → 연도 그룹핑 |
| 분석 | 평점-인기도 분포 / Top N 비교 | ScatterChart + BarChart | movies |
| 검색 | 키워드 기반 실시간 검색 | 카드 리스트 | TMDB Search API |

**필터 시스템**: 연도 범위 슬라이더, 장르 다중선택, 정렬 기준(평점/인기도/개봉일), Top N 개수 조절

### 5.2 KOFIC 박스오피스 대시보드 (7개 탭 × 3개 기간)

| 탭 | 시각화 | 데이터 소스 | 특징 |
|----|--------|-----------|------|
| 순위 | 박스오피스 순위표 | daily/weekly_box_office | 순위 변동 표시 (NEW/▲/▼) |
| 매출 | 영화별 매출 막대/타임라인 | daily/weekly_box_office | 당일 매출 + 누적 매출 |
| 관객 | 영화별 관객수 차트 | daily/weekly_box_office | 당일 관객 + 누적 관객 |
| 트렌드 | 월별/계절별 시장 추이 | weekly_box_office 집계 | 차트 + 테이블 + AI 해설 |
| 파생통계 | 효율 지표 (스크린당/회차당) | daily → DerivedStatsDto | 파이차트 + 바차트 + 테이블 |
| 추적 | 개별 영화 주차별 추이 | daily → MovieTrackingDto | 라인차트 + 바차트 + 테이블 |
| 역대순위 | 역대 관객/매출 Top 20 | daily → AllTimeRankingDto | 토글(관객/매출) + 바차트 |

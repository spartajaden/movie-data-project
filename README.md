# CineStats

영화 데이터를 수집하고 다양한 시각으로 분석하는 대시보드 프로젝트입니다.

TMDB에서 영화 정보를 가져와 장르 분포, 연도별 트렌드, 흥행 vs 평점 상관관계 등을 차트로 시각화합니다.

## 구조

```
data-miniproject/
├── front/   # React 기반 프론트엔드
└── back/    # 데이터 수집 및 API 서버 (예정)
```

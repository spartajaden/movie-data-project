# 사용자요구사항정의서

## 1. 문서 개요

| 항목       | 내용                                                                                                                                                                            |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 프로젝트명 | CineStats — 영화 데이터 대시보드                                                                                                                                                |
| 버전       | 1.0                                                                                                                                                                             |
| 작성일     | 2026-06-20                                                                                                                                                                      |
| 목적       | TMDB 및 KOFIC(영화진흥위원회) 오픈 API에서 수집한 영화 데이터를 시각화하고, AI 기반 분석 코멘터리를 제공하며, 게이미피케이션 요소를 통해 사용자 참여를 유도하는 웹 애플리케이션 |

---

## 2. 시스템 구성

### 2.1 기술 스택

| 구분         | 기술                                                              |
| ------------ | ----------------------------------------------------------------- |
| 프론트엔드   | React 19, TypeScript, Vite, Tailwind CSS, Recharts, Framer Motion |
| 백엔드       | Spring Boot 3.5, Spring Data JPA, Lombok                          |
| 데이터베이스 | MySQL 8.0 (운영), H2 (개발/테스트)                                |
| AI           | Ollama (Qwen 3.5:4b 로컬 LLM)                                     |
| 외부 API     | TMDB API, KOBIS(KOFIC) 오픈 API                                   |

### 2.2 시스템 구성도

```
[사용자 브라우저]
    │
    ├── http://localhost:5173  (Vite Dev Server)
    │       └── React SPA
    │
    └── http://localhost:8080  (Spring Boot)
            ├── REST API (/api/**)
            ├── Board API (/board/**)
            ├── TMDB API 연동
            ├── KOBIS API 연동
            └── Ollama API 연동 (localhost:11434)
```

## 10. 제약사항 및 전제조건

| 구분            | 내용                                                              |
| --------------- | ----------------------------------------------------------------- |
| 외부 API 의존성 | TMDB API와 KOBIS API의 가용성에 따라 데이터 수집이 제한될 수 있다 |
| AI 기능         | Ollama가 로컬에서 실행 중이어야 AI 코멘터리 기능이 동작한다       |
| 데이터 보존     | 일별 박스오피스 데이터는 최근 30일만 보존한다                     |
| 인증 미구현     | 현재 버전에서는 사용자 인증/권한 관리가 구현되어 있지 않다        |
| API 키 관리     | TMDB, KOBIS API 키는 별도 발급이 필요하다                         |
| 실행 환경       | Java 17 이상, Node.js 20 이상 (64비트) 환경이 필요하다            |
